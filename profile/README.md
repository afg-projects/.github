# 灵蛙 AFG Projects

[![GitHub Org](https://img.shields.io/badge/GitHub-afg--projects-blue?logo=github)](https://github.com/afg-projects)

AFG（Agile Frog）是一个面向企业场景的开源平台工程，致力于构建高质量的企业级开发框架和应用平台。

## 🚀 核心项目

| 项目 | 描述 | 技术栈 |
|------|------|--------|
| [afg-framework](https://github.com/afg-projects/afg-framework) | 企业级 Java 开发框架 | Java 25, Spring Boot 4 |
| [afg](https://github.com/afg-projects/afg) | 灵蛙企业级应用平台 | 微服务架构 |

## ✨ 特性

- **企业级框架** - 基于 Spring Boot 4 构建的现代化 Java 开发框架
- **模块化设计** - 核心功能、数据访问、中间件集成独立模块
- **开箱即用** - Spring Boot Starter 自动配置，快速集成
- **高测试覆盖** - 95%+ 测试覆盖率，质量有保障

## 📦 Maven Central

```xml
<dependency>
    <groupId>io.github.afg-projects</groupId>
    <artifactId>afg-framework-core</artifactId>
    <version>1.0.0</version>
</dependency>
```

# 行为准则

## 🤝 我们的承诺

`afg-projects` 致力于成为一个开放、尊重、协作的开源社区。无论你是维护者、贡献者还是使用者，都应该能在这里安全地参与讨论和贡献。

## ✅ 我们鼓励的行为

- **尊重他人** - 保持耐心与克制
- **聚焦问题** - 讨论集中在代码、设计和技术本身
- **乐于助人** - 帮助新贡献者理解项目
- **证据导向** - 以复现、验证为基础推进讨论
- **开放心态** - 接受建设性批评和建议

## ❌ 不可接受的行为

- 人身攻击、羞辱、歧视、骚扰
- 威胁、辱骂、恶意引战
- 故意破坏讨论秩序
- 未经允许公开他人隐私信息
- 其他明显不当行为

## 📍 适用范围

本行为准则适用于：

- GitHub Issues
- Pull Requests
- Discussions
- 与 `afg-projects` 相关的社区互动

## ⚖️ 执行方式

维护者有权对违反本准则的内容进行：

- 编辑或隐藏
- 关闭或拒绝
- 在严重情况下限制参与权限

## 📧 举报方式

如遇到严重问题：

1. 通过 GitHub 私信联系维护者
2. 或通过 SECURITY.md 中的渠道反馈

避免在公开讨论区扩大影响。

# `afg-projects/.github` 组织配置

这个仓库包含 afg-projects GitHub 组织的默认社区文件和配置。

## 📁 目录结构

```
.github/
├── profile/
│   └── README.md          # 组织主页展示内容
├── .github/
│   ├── ISSUE_TEMPLATE/    # Issue 模板
│   └── pull_request_template.md  # PR 模板
├── CODE_OF_CONDUCT.md     # 行为准则
├── CONTRIBUTING.md        # 贡献指南
├── SECURITY.md            # 安全策略
└── README.md              # 本文件
```

## 🏢 组织项目

| 项目 | 说明 |
|------|------|
| [afg-framework](https://github.com/afg-projects/afg-framework) | 企业级 Java 开发框架 |
| [afg](https://github.com/afg-projects/afg) | 灵蛙企业级应用平台 |

## 📝 文件说明

| 文件 | 用途 |
|------|------|
| `profile/README.md` | 组织主页展示内容 |
| `CODE_OF_CONDUCT.md` | 社区行为准则 |
| `CONTRIBUTING.md` | 如何参与贡献 |
| `SECURITY.md` | 安全漏洞报告方式 |
| `.github/ISSUE_TEMPLATE/` | Issue 模板（Bug、功能请求、问题） |
| `.github/pull_request_template.md` | Pull Request 模板 |
